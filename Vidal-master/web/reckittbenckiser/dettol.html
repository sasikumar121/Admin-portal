Дополнительная информация

Важные характеристики:

  • Удобный и компактный - для промывания раны не требуется воды, спрей можно использовать в любом месте
  • Спрей-нанесение - нет необходимости дотрагиваться до раны, что особенно удобно для детей
  • Содержит бензалкония хлорид - не пачкает кожу и одежду, эффективен против различных видов бактерий
  • Действует моментально - антибактериальное действие начинается в течение 60 секунд после нанесения *

* RECKITT BENCKISER (2006) In vitro study using European Standard
EN1276. Data on file.

Эффективность:

В исследованиях in vitro спрея для промывания ран Деттол
Бензалкония хлорид Спрей для ран в условиях загрязнения
была доказана способность спрея уничтожать различные виды
бактерий (тест 1, этапа 2, фазы CEN TC2161¹; EN 12762²)
и грибков (EN 16503³).

Бактерии:

  • Escherichia coli
  • Enterococcus hirae
  • Klebsiella pneumoniae
  • Proteus vulgaris
  • Pseudomonas aeruginosa
  • Staphylococcus aureus (частый возбудитель раневых инфекций)
  • Staphylococcus epidermidis
  • Streptococcus pyogenes
  • Устойчивые к метициллину Staphylococcus aureus (MRSA)
  • Устойчивые к ванкомицину Enterococcus faecalis (VRE)

Грибки:

  • Candida albicans

¹ RECKITT BENCKISER (1996) In vitro study using CEN TC216 Phase 2, Step 1 test methodology. Data on file.
² RECKITT BENCKISER (2006) In vitro study using European Standard EN1276. Data on file.
³ RECKITT BENCKISER (2006) In vitro study using European Standard EN1650. Data on file.
